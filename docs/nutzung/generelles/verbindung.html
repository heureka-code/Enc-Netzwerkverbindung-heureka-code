Netzwerkverbindung¶

Basisklasse¶

Für Klassen, über die gesendet und empfangen werden kann, wird eine abstrakte Basisklasse erstellt.

class enc_netzwerkverbindung._generell.verbindung.BasisCOM¶

Basisklasse fuer eine Netzwerkverbindung

abstract receive() → bytes¶

Empfaengt eine Nachricht und liefert diese zurueck

abstract send(data: bytes)¶

Sendet eine Nachricht

Parameter

data – Die zu sendende Nachricht

EncVerbindung¶

Eine Implementierung der Basisklasse verwendet AES für eine verschlüsselte Verbindung.

class enc_netzwerkverbindung._generell.EncVerbindung(verbindung: BasisCOM, aes_kodierung)¶

Verwaltet eine verschluesselte Verbindung mit AES

receive() → bytes¶

Wartet auf eine verschluesselte Nachricht und liefert diese zurueck

send(data: [<class 'str'>, <class 'bytes'>])¶

Sendet eine verschluesselte Nachricht an den Client

Parameter

data – Die zu sendende Nachricht

Sockets¶

Um die für Client und Server verschiedene Verwendung von Sockets zu abstrahieren werden zusätzlich zwei Klassen implementiert.

Client¶

Für den Client gibt es eine Implementierung, die den Socket verwaltet. Die Nachrichten werden hierbei allerdings nicht verschlüsselt.

class enc_netzwerkverbindung._generell.verbindung.ClientVerbindungsVerwalter(host: str, port: int)¶

Verwaltet die Clientseitige Netzwerkverbindung

receive() → bytes¶

Wartet auf eine Nachricht des Servers und liefert diese zurueck

send(data: bytes)¶

Sendet Daten an den Server

Parameter

data – Die zu sendende Nachricht

Server¶

Das Gegenstück zum eben gezeigten. Auch hier wird (noch) nicht verschlüsselt.

class enc_netzwerkverbindung._generell.verbindung.ServerVerbindungsVerwalter(wfile, rfile)¶

Verwaltet die Serverseitige Netzwerkverbindung

receive() → bytes¶

Wartet auf eine Nachricht des Clients und liefert diese zurueck

send(data: bytes)¶

Sendet Daten an den Client

Parameter

data – Die zu sendende Nachricht

IP-Adresse¶

Ermittelt die IP-Adresse des Computers
